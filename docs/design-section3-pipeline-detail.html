Extract：文本 → Claims + Evidence + 数值声明

输入 / 输出

class ExtractionResult(BaseModel): """Extract 阶段的完整输出""" claims: list[Claim] # ≥ 5 条 evidence_items: list[Evidence] # ≥ 5 条 numerical_statements: list[NumericalStatement] protocol: Protocol # 研究方法描述 limitations: list[Limitation] # 论文自述或 Agent 识别的局限 class Claim(BaseModel): id: str # "C-001", "C-002", ... text: str # 结论原文或近原文表述 type: ClaimType # experimental_result | computational | review_statement status: ClaimStatus = "unverified" # unverified → supported | contradicted | partial extraction_method: ExtractionMethod # ★ provenance 核心字段 source: SourceLocation # 页码 + 段落引用 evidence_refs: list[str] = [] # ["E-001", "E-002"] class ExtractionMethod(str, Enum): EXACT_QUOTE = "exact_quote" # 原文直接引用 → 最高可信度 LLM_INFERRED = "llm_inferred" # 模型推断/概括 → 需标记 REVIEW_REQUIRED = "review_required" # 无法确定 → 必须人工复核 class SourceLocation(BaseModel): page: int # 页码 section: str # "Results, Section 3.2" quoted_text: str # 原文引用片段 figure_ref: str | None # "Figure 3" 或 None table_ref: str | None # "Table S2" 或 None class NumericalStatement(BaseModel): id: str # "N-001" claim_id: str # 关联的 claim formula_nl: str # 自然语言公式："λ = √(D/D*)" parameters: dict[str, float] # {"D": 7.6e-6, "D_star": 2.9e-6} reported_value: float # 论文报告值 unit: str # "cm²/s" source: SourceLocation # 出处

LLM Prompt 结构（核心）

# prompts/extract_claims.md You are an ECS literature extraction specialist. Input: structured paper text with page numbers. Extract: 1. Claims — at least 5 key scientific conclusions. For each claim, you MUST specify: - extraction_method: "exact_quote" if you can quote the paper verbatim; "llm_inferred" if you are summarizing or interpreting; "review_required" if the text is ambiguous. 2. Evidence — for each claim, the supporting source: page number, section name, figure/table reference, and the exact quoted text. 3. Numerical statements — any quantitative relationship that can be computationally verified (equations, reported values with units, statistical results). CRITICAL RULES: - NEVER fabricate data, references, or values. - If a claim lacks clear evidence, set extraction_method= "review_required". - If a numerical statement is missing parameters for verification, mark it explicitly. - Distinguish between what the paper STATES vs what you INFER. Inference MUST be tagged as "llm_inferred".

错误处理

Claims 少于 5 条 → 标记 review_required，用 LLM 二次扫描未被覆盖的章节，仍不足则记录到 limitations
某 Claim 缺 Evidence → extraction_method = "review_required"，不编造证据
LLM 返回了不在原文中的内容 → Validate 阶段会交叉校验 quoted_text 是否在原文中出现（子串匹配）

错误处理

公式转译失败（LLM 无法转为 Python）→ calculation_error，保留原始 formula_nl 供人工审查
参数不足（论文只给了方向没给数值）→ insufficient_data，建议 review_required

Hallucination 检测算法

def check_quoted_text_in_source( quoted: str, page_text: str, threshold: float = 0.6 ) -> bool: """检查 quoted_text 是否在原文中出现 1. 先做精确子串匹配 2. 如精确匹配失败，做 fuzzy 匹配（token overlap） 3. fuzzy 匹配 < threshold → 标记为潜在 hallucination """ if quoted in page_text: return True quoted_tokens = set(quoted.lower().split()) page_tokens = set(page_text.lower().split()) overlap = len(quoted_tokens & page_tokens) ratio = overlap / len(quoted_tokens) return ratio >= threshold
